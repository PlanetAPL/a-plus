Footnote

On the RS6000, the workspace size cannot be increased when a mapped file exists for an A+ process, because A+ requires that the entire workspace be contiguous and AIX requires that all space be taken from one end of the address space; thus, the mapped file blocks enlargement of the workspace. Space must be made available before any file is mapped; use either the -w A+ command line parameter (see "Invocation and Environment") or $wa.
